## Full description

LedgerBrain / LedgerCore provides x402 pay-per-use crypto compliance reports for AI agents
and API clients. Agents can run KYA address risk reports before sending funds to a receiving
address, KYT transaction risk reports when receiving or reviewing a payment, and KYE entity
intelligence reports for counterparty due diligence context. Reports are paid per use with USDC
on Base mainnet through x402. No LedgerBrain API key is required for x402 endpoints. Compliance
risk screening includes AML/sanctions-related risk context where available in the report.
Reports are compliance intelligence outputs, not legal advice.
