## Short description

LedgerBrain / LedgerCore provides x402 pay-per-use KYA, KYT, and KYE crypto compliance reports for AI agents.

## Long description

LedgerBrain / LedgerCore provides x402 pay-per-use crypto compliance reports for AI agents and API clients. Agents can run KYA address risk reports before sending funds to a receiving address, KYT transaction risk reports when receiving or reviewing a payment, and KYE entity due diligence reports for counterparty context. Reports are paid per use with USDC on Base through x402. No LedgerBrain API key is required for x402 endpoints.
